# Code of Conduct

We want this project to be welcoming and safe for everyone.

## Be respectful

- Assume good intent and be kind.
- Keep discussions focused on technical topics and constructive outcomes.
- Harassment, discrimination, and personal attacks are not tolerated.

## Reporting

If something happens that violates this code of conduct, report it privately:

- Email: `conduct@fluxtopus.com`

We will review reports promptly and take appropriate action.
